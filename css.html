CSS corresponde a las siglas de Cascade Style Sheet u Hojas de Estilos en Cascada y como indica su nombre CSS3 corresponde a la tercera revisión del mismo CSS.

CSS se encarga de decirle al navegador cómo debe mostrar un elemento concreto: estilo, espaciado y posición. A diferencia de JavaScript, CSS no es un lenguaje de programación ni tampoco es uno de etiquetas como HTML. No se puede comparar con nada de esto.

Es utilizado para aplicar estilos al HTML y situarlo en la página web. Se puede utilizar dentro de la página HTML o en un fichero propio con extensión .css. Lo que se utiliza más es esto último la hoja de estilos separada) ya que este lenguaje fue creado para no mezclar los estilos con el lenguaje HTML. Esto es así porque es más fácil mantener.

Ejemplo: 

        selector {
           propiedad1:valor;
           propiedad2:valor;
           propiedad3:valor;
        }
