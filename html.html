Elemento img

Cuando una imagen es significativa para el documento, es decir, aporta significado al contenido ya sea la imagen de un autor, la imagen de una noticia o un gráfico se debe añadir a la página HTML con el elemento img.

Los atributos que posee dicho elemento se describen a continuación:

alt: este atributo es necesario ya que aporta una descripción de la imagen en el caso que ésta no esté disponible porque el navegador no acepta imágenes, porque no la puede encontrar, etc. Además, las personas que tienen alguna discapacidad visual utilizan tecnologías de asistencia que leerán el contenido del atributo alt, por lo que hace que sea importante el contenido de dicho atributo.

title: este atributo no da seguridad a la hora de dar información sobre la imagen. Pero, en cambio, es bueno para describir aclaraciones o sensaciones que da la imagen.

En el siguiente fragmento de código se muestra un ejemplo del elemento img con los atributos comentados:

          
            <!DOCTYPE html>
            <html>
              <head>
                <title>Ejemplo del elemento img de HTML</title>
              </head>
              <body>
                <img src="./img/imgHTML.jpg" alt="Esto es un ejemplo del uso del 
                    elemento img" title="Ejemplo del atributo title">
              </body>
            </html>

A continuación se muestra como se vería la imagen del ejemplo anterior, si se posiciona el ratón sobre la imagen se puede ver el contenido del atributo title que se ha descrito anteriormente.
